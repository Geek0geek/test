Head and Eye Tracking with MediaPipe

About the Project

This project uses MediaPipe and OpenCV to track facial landmarks, including the head and eyes, in real-time. The program also calculates and displays FPS during execution.

Instructions

  1. Ensure Python 3 is installed on your system.
  2. Install the required libraries using: 
    pip install opencv-python mediapipe
  3. Run the Python script tracking.py.
  4. Press q to quit the program.
